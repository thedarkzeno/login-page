## :construction_worker: How to run:

Clone the project from github:

```shell
git clone https://github.com/thedarkzeno/login-page
cd login-page
```

Install dependencies:

```shell
yarn
# or
npm i
```

Start the application:

```shell
yarn start
# or
npm start
```
